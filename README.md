# Jobie - Job Portal Website

## Description

Jobie is a responsive job portal website built using Bootstrap 5. It showcases various job opportunities with a clean, modern design. The site includes a navigation bar, image carousel, job listings displayed as cards, and a footer with contact information and social links.

## Features

- **Responsive Design**: Utilizes Bootstrap's grid system for mobile-first, responsive layouts.
- **Navigation Bar**: Includes links to Home, About, Jobs (with dropdown for IT Jobs, Govt Jobs, Internships), and Messages. Also features a search bar.
- **Image Carousel**: Displays featured images at the top of the page.
- **Job Listings**: Presents job opportunities in card format, each with an image, title, skills required, location, and an "Apply" button.
- **Footer**: Contains contact details, a subscription form, and social media links.

## Technologies Used

- HTML5
- CSS3 (with custom styles)
- Bootstrap 5.3.3
- JavaScript (Bootstrap Bundle for interactive components)

## How to Run

1. Ensure you have a modern web browser installed (e.g., Chrome, Firefox, Edge).
2. Download or clone the project files to your local machine.
3. Open the `webpage.html` file in your web browser.
4. The website should load and display the job portal interface.

## Project Structure

- `webpage.html`: Main HTML file for the Jobie job portal.
- `web2.html`: Additional demo HTML file showcasing Bootstrap grid layouts.
- `logo.png`: Logo image used in the navigation bar.
- Image files (`1.webp`, `2.avif`, `3.png`, `c1.jpg`, `c2.avif`, `c3.png`, `c4.jpg`, `c5.avif`, `c6.jpg`): Assets for the carousel and job cards.
- `TODO.md`: Project todo list (if applicable).
